IT Inventory Agent

Покрокова інструкція для компіляції агента у виконуваний файл .exe та підготовки до розповсюдження

📋

Передумови

Компонент Версія Статус
🐍 Python 3.10 — 3.12 ● Обов'язково
📦 pip Актуальна ● Обов'язково
🔧 PyInstaller ≥ 6.0 ● Обов'язково
🎨 icon.ico — ○ Опціонально
⚡

Крок 1 — Створення віртуального оточення

Відкрийте cmd.exe у каталозі agent/ та створіть віртуальне оточення:

cmd 
python -m venv venv

Активуйте віртуальне оточення:

cmd 
venv\Scripts\activate
📌
Після активації на початку рядка з'явиться (venv). Всі наступні команди виконуйте тільки з активованим оточенням.

Встановіть залежності проєкту та PyInstaller:

cmd 
pip install -r requirements.txt
pip install pyinstaller

Перевірте, що PyInstaller встановлений коректно:

cmd 
pyinstaller --version
⚠️
PowerShell: Якщо використовуєте PowerShell замість cmd, команда активації інша: venv\Scripts\Activate.ps1
Може знадобитись дозвіл: Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
🔨

Крок 2 — Компіляція

Відкрийте cmd.exe у каталозі agent/ та виконайте:

cmd 
pyinstaller --name "ITInventoryAgent" --onefile --windowed --icon=icon.ico --add-data "agent.py;." --hidden-import=psutil --hidden-import=requests --hidden-import=win32api --hidden-import=win32com --hidden-import=wmi agent_gui.py
⚠️
Важливо: Команда призначена для cmd.exe. У PowerShell працюватиме так само, якщо все в одному рядку (без символу ^).
Параметр Що робить
--name Ім'я вихідного .exe файлу
--onefile Все пакується в один файл (без папки з DLL)
--windowed Без чорного вікна консолі (GUI-додаток)
--icon Іконка для .exe файлу (формат .ico)
--add-data Включає agent.py як ресурс (GUI імпортує його)
--hidden-import Примусово додає модулі, які PyInstaller не знаходить автоматично
⏳

Крок 3 — Результат

Компіляція займе 1–3 хвилини. Готовий файл буде в папці:

шлях 
agent\dist\ITInventoryAgent.exe

Також будуть створені папки build/ та файл .spec — їх можна видалити.

📁

Структура каталогу для запуску

Скопіюйте ці файли в одну папку на цільовому комп'ютері:

📂 ITInventoryAgent/
├── ✅ ITInventoryAgent.exe Скомпільований агент
├── ⚙️ .env Конфігурація підключення до сервера
├── ⚙️ gui_settings.json Налаштування GUI (авто при першому запуску)
├── 🎨 icon.ico Іконка для системного трею
└── 📄 .env.example Шаблон конфігурації (опціонально)
Файл Статус Опис
ITInventoryAgent.exe ● Обов'язково Головний виконуваний файл з GUI та збором даних
.env ● Обов'язково Адреса сервера, логін, пароль, розташування ПК
gui_settings.json ○ Авто Створюється автоматично. Автозапуск, трей, авто-сканування
icon.ico ○ Опціонально Іконка для трею. Без неї — стандартна іконка
⚙️

Крок 4 — Налаштування .env

Відредагуйте файл .env на кожному комп'ютері:

.env 
# URL серверу інвентаризації (без / на кінці)
API_URL=http://192.168.1.100:8000

# Логін та пароль користувача
USERNAME=agent
PASSWORD=agent_password

# Розташування цього комп'ютера
LOCATION=Офіс, кімната 101

# Інтервал збору в секундах (для автоматичного режиму)
INTERVAL=3600
🔴
Без файлу .env агент не зможе підключитися до сервера!
Переконайтесь, що .env знаходиться в тій самій папці, що й ITInventoryAgent.exe
🚀

Крок 5 — Запуск

Запустіть ITInventoryAgent.exe подвійним кліком.

Агент з'явиться у системному треї (біля годинника).

Відкрийте головне вікно кліком по іконці в треї.

⚠️
Перший запуск може бути повільнішим — Windows SmartScreen перевіряє файл. Натисніть «Додаткові відомості» → «Виконати все одно».
🔧

Усунення проблем

Проблема Рішення
❌ Антивірус блокує EXE Додайте папку агента у виключення антивірусу. PyInstaller-збірки часто хибно детектуються
❌ Failed to execute script Запустіть з консолі — побачите помилку. Зазвичай це відсутній .env
❌ Не збирає дані WMI Запустіть від імені Адміністратора (ПКМ → «Запустити як адміністратор»)
❌ ModuleNotFoundError Додайте --hidden-import=назва та перезберіть
❌ Великий розмір (>100 МБ) PySide6 включає багато Qt-бібліотек. Див. оптимізовану збірку нижче
❌ Папка logs/ не створюється Переконайтесь, що агент має права на запис у свою папку
💡

Оптимізована збірка (менший розмір)

Для зменшення розміру EXE виключіть зайві Qt-модулі:

cmd 
pyinstaller --name "ITInventoryAgent" --onefile --windowed --icon=icon.ico --add-data "agent.py;." --hidden-import=psutil --hidden-import=requests --hidden-import=win32api --hidden-import=win32com --hidden-import=wmi --exclude-module=QtWebEngine --exclude-module=QtWebEngineCore --exclude-module=Qt3D --exclude-module=QtCharts --exclude-module=QtDataVisualization --exclude-module=QtMultimedia --exclude-module=QtQuick --exclude-module=QtQml agent_gui.py

Це може зменшити файл з ~120 МБ до ~60–80 МБ.

📦
